Halik recorder - IntelliJ plugin

Halik can be installed as every other IntelliJ plugin. If you have never done it please follow the instructions below.

Installation instructions

1. Go to "Preferences", find "Plugins" panel, and click on "Browse repositories".
2. Type "Halik", and click "Install" when it shows up in the view.
